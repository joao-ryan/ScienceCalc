# 🚀 Sobre o Projeto

**ScienceCalc** é uma **calculadora científica moderna** desenvolvida com **React**, focada em oferecer uma experiência de cálculo poderosa, visual e responsiva.

O projeto foi construído aplicando **boas práticas de engenharia de software**, arquitetura modular e componentes reutilizáveis.

O objetivo é criar uma ferramenta matemática moderna que combine:

- 🧠 cálculo científico  
- 📈 visualização gráfica  
- 📏 conversão de unidades  
- 📜 histórico de cálculos

## 📈 Visualização de Gráficos

A aplicação permite visualizar funções matemáticas em **gráficos interativos**.

Exemplos de funções:

```text
y = x²
y = sin(x)
y = cos(x)
y = log(x)
```

Recursos:

- Renderização dinâmica
- Escala automática
- Visualização matemática

---

## 📏 Conversor de Unidades

Sistema integrado de conversões:

### Comprimento

- metro → quilômetro
- centímetro → metro

### Massa

- grama → quilograma

### Temperatura

- Celsius → Fahrenheit

---

## 📜 Histórico de Cálculos

A calculadora mantém um histórico das operações realizadas.

Permite:

- revisar cálculos
- acompanhar operações anteriores
- melhorar produtividade

# 🏗 Arquitetura do Projeto

Estrutura baseada em **componentização e separação de responsabilidades**.

```text
src
│
├── components
│   ├── Calculator.jsx
│   ├── Display.jsx
│   ├── Keypad.jsx
│   ├── GraphPanel.jsx
│   ├── Converter.jsx

│
├── hooks
│   └── useCalculator.js
│
├── engine
│   ├── mathEngine.js
│   └── conversions.js
│
├── utils
│   └── constants.js
│
├── styles
│   └── calculator.css
│
└── App.jsx
```

# 🧑‍💻 Instalação

Clone o repositório:

```bash
git clone https://github.com/seu-usuario/sciencecalc
```

Entre na pasta:

```bash
cd sciencecalc
```

Instale as dependências:

```bash
npm install
```

Execute o projeto:

```bash
npm run dev
```
